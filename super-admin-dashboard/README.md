# Super Admin Dashboard - Teacher Attendance Module

A comprehensive administrative interface for managing teacher attendance through QR code scanning.

## Features

- Generate and download QR codes for teacher attendance
- Real-time attendance monitoring
- Teacher attendance history tracking
- Modern Material-UI design with enhanced styling
- Responsive layout for all device sizes
- Direct API integration with backend

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3000`

## Usage

1. Navigate to the "Teacher Attendance" section in the sidebar
2. Generate and download the QR code for teachers
3. Monitor real-time attendance status
4. View detailed attendance history for each teacher

## API Integration

The application connects to the backend API at `http://localhost:5000` for attendance data storage and retrieval.

## Technologies Used

- React 18
- TypeScript
- Material-UI v7
- Vite
- qrcode.react for QR code generation
- Modern CSS with gradients and animations
